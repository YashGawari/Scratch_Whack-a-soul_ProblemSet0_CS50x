# Scratch_Whack-a-soul_ProblemSet0_CS50x
A Scratch based Whack a Mole style game created for CS50 Problem Set 0.

Project Overview - This is a whack-a-mole style game built in scratch in which the player earnc points by clicking on the soul when it pops out of a hole.

Installation/Setup - 
  To run this game, you need one of the following:
  
  Option 1: Web Browser (Recommended)
  Any modern browser (Chrome, Edge, Firefox)
  Option 2: Scratch Desktop (Offline)
  Scratch Desktop installed on your system

Controls - Just simple mouse left clicks.
